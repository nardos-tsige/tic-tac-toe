# ⚡ Tic Tac Toe – Solo or Duo!

Modern, animated Tic Tac Toe game with **Smart AI**, 2-player local mode, score tracking, and responsive design.

## 🎮 Features
- 🤖 Smart AI (Win/Block/Priority moves)
- 👥 Local Two-Player Mode
- ✨ Custom player names
- 🏆 Scoreboard & animations
- 🎨 Glowing effects, toast notifications
- 📱 Fully responsive (mobile friendly)

## 🛠️ Tech Stack
- HTML5, CSS3 (Flex/Grid)
- JavaScript (ES6+)
- Font Awesome Icons
- Google Fonts

## 🚀 How to Run
1. Download all files (`index.html`, `style.css`, `script.js`)
2. Open `index.html` in any modern browser
3. Enjoy!

## 📁 Folder Structure
tic-tac-toe/
├── index.html
├── style.css
├── script.js
└── README.md

## 🧠 SOLID Principles Applied
- Single Responsibility: each function handles one task (checkWin, AI move, render)
- Open/Closed: easy to extend difficulty levels
- Dependency Inversion: game logic independent from DOM
